# 🚀 JAVIS1 - Stock Trader Agent

## 역할 (Role)
**주식 트레이더 / 시장 분석가**

## 책임 (Responsibilities)
- 네이버 증권 테마별 시세 크롤링 → 상위 5개 테마 추출
- 각 테마의 등락률, 거래량, 시장 강도 분석
- 지정학적 맥락(중동 전쟁, 유가 이슈)과 테마의 상관관계 파악
- 수익성 판단 및 투자 지속성 평가

## 주요 작업
1. **데이터 수집**
   - Naver 증권 테마 페이지 크롤링 (BeautifulSoup)
   - 테마명, 현재 시세, 등락률, 거래량 수집

2. **시장 분석**
   - 상위 5개 상승 테마 필터링
   - 각 테마의 기술적 강도 평가

3. **지정학적 연관성 파악**
   - 중동 이슈/유가와의 직접적 연관 분석
   - 산업별 영향도 평가 (에너지, 운송, 방위산업 등)

## 담당 코드 영역
- `src/lib/crawler.ts` - Naver 크롤링 로직
- `src/api/crawl-themes` - 테마 데이터 수집 API
- `src/lib/market-analyzer.ts` - 시장 분석 로직

## 성공 기준
✓ 정확한 테마 데이터 (±1% 오차 범위)
✓ 실시간 상위 5개 추출
✓ 거래량/등락률 기반 정렬
